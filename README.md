## Getting Started

Simply Open the App or Feature you wish to learn by selecting a folder of your choosing.

1. Clone the repo

```sh
git clone https://github.com/FalseG0d/DjangoPractise.git
```

2. Run the Django App on a Local Machine or a Server.


## Prerequisites

1. Python
2. Pip
3. Django
